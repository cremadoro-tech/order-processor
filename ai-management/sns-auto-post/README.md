# SNS自動投稿システム

商品ごとの写真とURLを管理し、SNS投稿を自動化するシステム。

## セットアップ手順

### 1. Google Sheetsを作成
1. [Google Sheets](https://sheets.google.com) で新しいスプレッドシートを作成
2. 名前を「SNS投稿管理」に変更

### 2. Google Apps Scriptをデプロイ
1. スプレッドシートのメニューから **拡張機能 → Apps Script** を開く
2. `gas/Code.gs` の内容をエディタに貼り付け
3. **プロジェクトの設定 → スクリプトプロパティ** で以下を追加:
   - `CLAUDE_API_KEY`: AnthropicのAPIキー
4. スプレッドシートに戻ってページを再読み込み

### 3. シートを初期化
1. メニューバーに「SNS投稿」メニューが表示される
2. **SNS投稿 → シートを初期化** を実行
3. 以下の3シートが自動作成される:
   - **商品マスタ**: 商品情報の管理
   - **投稿スケジュール**: 投稿計画と生成テキスト
   - **テンプレート**: SNS別の投稿ルール（初期データ入り）

### 4. 運用フロー
1. 「商品マスタ」に商品を登録
2. 「投稿スケジュール」に投稿予定を入力（SNS・商品ID・形式を指定）
3. **SNS投稿 → 選択行の投稿文を生成** or **未生成の投稿文を一括生成**
4. 生成されたテキストを確認・編集して各SNSに手動投稿（Phase 1）

```
    └── appsscript.json # GASマニフェスト
```
