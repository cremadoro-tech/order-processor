# /rakuten-lp — 楽天市場商品LP・HTMLを自動生成する

## 概要

NotebookLMで整理した楽天市場の公式ガイドラインに基づき、コンバージョン最大化を目的とした商品LP（スマートフォン用商品説明文）をHTML形式で自動生成するスキル。
ページ構成・SEO・デザイン・コピーライティング・信頼性要素・禁止事項チェックを一括で担保する。

---

## インプット形式

```
/rakuten-lp
商品名: [商品名・型番]
カテゴリ: [印鑑 | 筆記具 | 文具]
価格: [円]
ターゲット: [例: 30〜50代女性、オフィスワーカー等]
主な特徴:
  - [特徴・訴求ポイント 1]
  - [特徴・訴求ポイント 2]
  - [特徴・訴求ポイント 3]
競合URL: （任意）
  - [競合商品ページURL 1]
  - [競合商品ページURL 2]
権威性バッジ: （任意。例: 楽天ランキング1位（2026年3月・文具カテゴリ））
レビュー抜粋: （任意。実際のお客様の声を1〜3件）
カラーイメージ: （任意。例: ネイビー×ゴールド、ミニマル白基調等）
```

**必須**: 商品名・カテゴリ・価格・主な特徴
**任意**: ターゲット・競合URL・権威性バッジ・レビュー抜粋・カラーイメージ

## 実行フロー

### Step 1: インプット確認・補足質問

不足情報がある場合のみ最小限の質問を行い、すぐに生成フェーズへ移行する。
競合URLが指定された場合はWebFetchで内容を取得してベネフィット訴求の参考にする。

### Step 2: 商品名・キャッチコピー生成

**商品名（全角127文字以内）:**
- 検索キーワードは冒頭19〜30文字以内に配置（左寄せ）
- 構成: ジャンルキーワード ＋ 商品名 ＋ 仕様 ＋ 色・サイズ
- バリエーション情報（色・サイズ）は商品属性情報に任せ、商品名に過剰に詰め込まない

**キャッチコピー（全角30文字以内）:**
- 「送料無料」「ポイントxx倍」「xx%OFF」等の販促文言を集約
- 商品名に入れられない補足キーワードを記載

### Step 3: HTML生成（セクション順厳守）

以下の順序でHTMLを生成する。

#### Sec1. ファーストビュー
- 商品の全体像がわかるメインビジュアルスペース
- 最大のベネフィットを端的に示すコピー
- 権威性バッジ（楽天ランキング・累計販売数等 ※根拠必須）

#### Sec2. 安心情報・基本情報
- 送料（送料無料条件）
- 納期（あす楽対応等）
- 返品交換保証

#### Sec3. 興味喚起とベネフィット
- 具体的な使用シーン・ターゲット層の利用イメージ
- サイズ感の比較
- 「こんなお悩みありませんか？」等の共感と解決策

#### Sec4. 説得と根拠・他社比較
- 素材・成分・独自構造のディテール詳細
- 従来品や他社製品との比較・客観的なデータ

#### Sec5. 不安解消と信頼構築（トラスト）
- レビュー抜粋（お客様の声）
- よくある質問（FAQ）Q&A形式
- カラー・サイズの全バリエーション一覧と詳細スペック表

#### Sec6. クロージング・回遊導線
- クーポンや期間限定オファー
- 関連商品・セット商品・別カテゴリへの回遊リンク（店舗内リンクのみ）

### Step 4: 禁止事項チェック（自動）

生成したHTMLに対して以下の違反がないかパース・確認する:

- [ ] `<style>` `<link>` `<iframe>` `<script>` タグをスマホ用説明文に使用していない（インラインまたは`<table>`レイアウト使用）
- [ ] 外部リンク（自社サイト・SNS・他モール）のURL・QRコードが含まれていない
- [ ] 商品と無関係なブランド名・カテゴリ単語の羅列（ワードサラダ）がない
- [ ] 隠し文字（背景色と同色のテキスト・極小フォント）がない
- [ ] 「送料無料」「ポイントxx倍」等の販促文言が商品名に混入していない
- [ ] 「最安値」「No.1」「世界初」等の根拠のない最大級表現がない
- [ ] 薬機法・景表法違反の断定表現がない

違反があれば自動修正してから出力する。

### Step 5: 出力

完成したHTMLを出力し、以下を簡潔に提示する:
- 狙ったメインキーワード（3〜5語）
- 使用したデザインカラー（3色）
- 修正した禁止事項（あれば）

## デザイン原則

### スマホ最適化（最優先）

- 本文フォントは16px相当以上
- タップ領域は最低44×44px以上
- 幅指定は固定値（px）ではなく相対値（%）を使用
- 横スクロールが発生しないレイアウト

### 画像ガイドライン

- 画像内テキストは全体の20%以下
- 画像周囲の枠線・L字・帯状デザインは禁止
- 背景は白（#FFFFFF）または自然な写真背景のみ（ベタ塗りカラー背景禁止）
- GIFアニメーションはサムネイルに使用しない

### カラー設計

- メインカラー・サブカラー・アクセントカラー（CTA用）の3色を基本構成
- 視認性の高いコントラストを保つ

---

## SEOルール（セマンティックサーチ対応）

- キーワードの羅列（ワードサラダ）は避け、使用シーンや悩み解決の「文脈」として自然な文章内に配置する
- 類義語・ゆらぎキーワードを含める
- 狙いたい検索キーワードは見出しとテキスト内で複数回（自然な範囲で）出現させる

---

## コピーライティング原則

### ベネフィット訴求

- スペック（機能）ではなく、使うことで得られる「未来の体験」を具体的な数字で表現する

### 読みやすさ

- 重要情報は箇条書きや短い段落にまとめる
- アイコンや図解（画像指示）を適宜交える

### CTA・マイクロコピー

- 「今すぐ購入する」「サイズを選ぶ」等、次の行動が明確なテキストを使用
- CTAの直近に「送料無料」「返品保証」等のリスクリバーサルを配置する

---

## 信頼性・購買促進要素

- **楽天ランキング1位** を記載する場合は「集計日」と「対象ジャンル名」を必ずテキストで併記する
- 高評価レビューの抜粋はバナー的に視覚化する（高評価の強要・サクラは規約厳禁）
- メディア掲載実績や専門家推奨ロゴはファーストビュー付近に配置する
- クーポン・ポイント倍率など「今買うべき理由」をページ上部に目立つように配置する

```html
<!-- Sec1: ファーストビュー -->
<table width="100%" cellpadding="0" cellspacing="0">
  <tr>
    <td align="center" style="padding:20px 10px;">
      <img src="main-image.jpg" alt="商品名" width="90%" style="max-width:600px;">
      <p style="font-size:20px;font-weight:bold;color:#1a1a1a;">最大ベネフィットのコピー</p>
    </td>
  </tr>
</table>
<!-- ... 各セクション続く -->
```

CSSはインラインスタイルまたは`<table>`レイアウトのみ使用（`<style>`タグ禁止）。
