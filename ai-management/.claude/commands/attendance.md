# 勤怠打刻 - ジョブカン自動操作

ユーザーが以下のキーワードを入力したら、対応するジョブカン打刻スクリプトを実行してください。

## キーワードとアクション対応

| ユーザー入力 | 実行アクション |
|------------|-------------|
| 出勤 | 出勤打刻 |
| 退勤 | 退勤打刻 |
| 休憩 / 休憩開始 | 休憩開始打刻 |
| 終了 / 休憩終了 | 休憩終了打刻 |

## 実行手順

1. ユーザーの入力からアクションを判定する
2. Bashツールで以下のコマンドを実行する（バックグラウンドで実行してよい）：

```bash
cd ~/My\ WorkSpace/ai-management/scripts && node jobcan/index.js [アクション]
```

### 各アクションのコマンド例
- 出勤: `node jobcan/index.js 出勤`
- 退勤: `node jobcan/index.js 退勤`
- 休憩開始: `node jobcan/index.js 休憩開始`
- 休憩終了: `node jobcan/index.js 休憩終了`

## ブラウザの操作

**通常（セッション有効時）：** ブラウザが自動で開き、ボタンを押して閉じます。ユーザー操作不要。

**セッション切れの場合：** 以下のメッセージが表示される:
```
⚠️  ジョブカンのセッションが切れています。
   node ec/login-helper.js jobcan でログインし直してください。
```
→ ユーザーにターミナルで `node ec/login-helper.js jobcan` を実行するよう案内する。

## 実行後の確認

コマンド実行後、「✅ [アクション]の打刻が完了しました」と表示されれば成功です。
エラーが出た場合はエラーメッセージをユーザーに伝えてください。
