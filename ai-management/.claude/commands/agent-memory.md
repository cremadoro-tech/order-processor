# /agent-memory — 重要情報をメモリに保存する

## 概要
会話中の重要な情報・意思決定・ユーザーの方針をメモリファイルに永続化するスキル。
次回の会話でも文脈を引き継げるよう、適切なカテゴリに整理して保存する。

## 実行手順

1. **保存すべき情報を抽出する**
   会話から以下の種類の情報を識別する：
   - **意思決定**: 何を決めたか、その理由
   - **個人の方針・好み**: 好むスタイル・やり方・価値観
   - **事実情報**: 数値・データ・重要な事実
   - **ビジネス情報**: 取引先・商品・EC設定等

2. **カテゴリを判定して保存先を決める**
   | カテゴリ | 保存先 |
   |:--------|:------|
   | 事業方針・価値観 | `00_context/memories/business-policy.md` |
   | 個人の好み・スタイル | `00_context/memories/preferences.md` |
   | 意思決定ログ | `00_context/memories/decisions.md` |
   | 重要な事実・データ | `00_context/memories/facts.md` |
   | EC・商品情報 | `00_context/memories/ec-info.md` |

3. **ファイルに追記する**
   - 既存の内容と重複しないよう確認する
   - 日付を付与して追記する
   - 関連する既存情報があれば更新する

4. **保存内容をユーザーに報告する**
   - 何をどこに保存したかを簡潔に報告する

## メモリファイルフォーマット

```markdown
## YYYY/MM/DD

### [カテゴリ]
- [保存した情報]
- 背景: [なぜそう決めたか・なぜ重要か]
```
