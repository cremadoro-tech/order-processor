# /trend-check — Nintで文具・印鑑ジャンルのトレンドをリサーチする

## 概要
EC市場分析ツール「Nint」に自動ログインし、以下の対象ジャンルの売れ筋ランキング・競合情報・トレンドデータを取得して `output/trends/` に保存するスキル。

## 調査対象ジャンル

| ジャンル | 主な調査カテゴリ |
|:--------|:--------------|
| 印鑑・ハンコ・スタンプ | 印鑑（認印・銀行印・実印）、スタンプ（シャチハタ等）、スタンプ台、印鑑ケース |
| 筆記具 | ボールペン、万年筆、鉛筆・シャープペンシル、マーカー・蛍光ペン、ゲルインクペン |
| その他文具 | ノート・手帳、ファイル・バインダー、テープ・のり、カッター・はさみ、付箋・メモ |

## 前提条件
- MCP `playwright` が有効になっていること
- `.env` に Nint のログイン情報が設定されていること
  ```
  NINT_EMAIL=
  NINT_PASSWORD=
  ```

## 実行手順

1. **Nint にログインする**
   - MCP `playwright` を使用してブラウザを起動
   - https://www.nint.jp にアクセスしてログイン
   - 認証情報は `.env` の `NINT_EMAIL` / `NINT_PASSWORD` を使用

2. **印鑑・ハンコ・スタンプジャンルのデータを取得する**
   - カテゴリ：印鑑、スタンプ、はんこ 等で検索・絞り込み
   - 売れ筋ランキング TOP10 を取得（商品名・価格・順位・前週比）
   - 競合商品の出品状況・価格帯・レビュー数を確認
   - トレンドセクションで注目キーワードを収集

3. **筆記具ジャンルのデータを取得する**
   - カテゴリ：ボールペン、万年筆、鉛筆、筆記具 等で検索・絞り込み
   - 売れ筋ランキング TOP10 を取得
   - ブランド別シェア・価格帯分布を確認
   - 競合の新規出品・価格変更をチェック

4. **その他文具ジャンルのトレンドを調査する**
   - 文具全体のトレンドセクションを確認
   - 急上昇キーワード・注目カテゴリを取得
   - 季節性・イベント連動（入学シーズン等）の動向を確認

5. **レポートを生成して保存する**
   - 各ジャンルのデータを整理・比較分析する
   - 自社商品への影響・機会・脅威を分析する（SWOT視点）
   - 具体的なアクション候補を提案する
   - 保存先: `output/trends/YYYY-MM-DD-nint-report.md`
